Entity-relationship model

  • Entity: “thing” or “object” in the real world that is distinguishable from all other objects.

  • Entity set: a set of entities of the same type that share the same properties.

E-R Diagram

  • 分成两部分的矩形: 一个实体集，等同于关系。
  • 菱形: 一个联系集，可以认为是$\text{natural join}$的结果。
  • 没有分隔的矩形: 联系集的属性。
  • 线段: 将实体集连接到联系集。
  • 虚线: 将联系集属性连接到联系集。

剩下还有两种双线和双菱形，但是还没讲到所以暂时不放进来。
